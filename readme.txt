== Installation ==

Note : This plugin works with Elementor. Make sure you have [Elementor](https://wordpress.org/plugins/elementor/) installed.
From the WordPress Dashboard:

	1.	Login to your WordPress admin dashboard.
	2.	Navigate to Plugins > Add New.
	3.	In the search bar, type Turbo Elementor.
	4.	Locate Turbo Elementor in the search results and click Install Now.
	5.	After installation is complete, click Activate to enable the plugin.
